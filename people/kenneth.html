Kenneth Reitz is a backend software engineer that focuses on interface design, primarily with the Python language. He is passionate about APIs, web services, reducing cognitive overhead, and building quality abstractions around human problems. He is the author of a number of libraries in the Python ecosystem, including Requests: HTTP for Humans, Pipenv, and more.
